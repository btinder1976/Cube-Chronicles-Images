# Cube Chronicles Images

KDP-ready ebook covers, paperback full-wrap covers, and raw cover art for *The Cube Chronicles* by Jeremy Tinder.

## File formats

- Ebook covers: JPG, 1600 x 2560 pixels
- Paperback covers: print-ready PDF full wrap, 6 x 9 trim with 0.125-inch bleed and page-count-based spine width
- Raw art: PNG with no title or other typography
